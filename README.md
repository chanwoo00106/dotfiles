# dotfiles

chanwoo's dotfiles

## Used

```bash
~ $ touch .zshenv
~ $ echo "export ZDOTDIR=\"\$HOME/.config/zsh\"" >> .zshenv
~ $ git clone https://github.com/chanwoo00106/dotfiles.git ~/.config
```
